JavaScript (« JS » en abrégé) est un langage de programmation dynamique complet qui, appliqué à un document HTML, peut fournir une interactivité dynamique sur les sites Web.

JavaScript est d'une incroyable flexibilité. Vous pouvez commencer petit, principalement côté Front-End, avec des carrousels, des galeries d'images, des variations de mises en page et des réponses aux clics de boutons.

Avec plus d'expérience, vous serez en mesure de créer des jeux, des graphiques 2D et 3D animés, et même travailler côté Back-End avec des applications complètes de gestion liées à des bases de données.

Contrairement à l'HTML et le CSS qui sont nativement des langages de balisage, JavaScript est un vrai langage de programmation.

Par soucis de clarté et le nombre limité de cours dans la section PDD, nous le verrons du côté Front-End, ce qui permettra d'avoir un résultat visible immédiatement.

Nous mettrons donc notre javascript sous forme de scripts immédiatement intégrés dans l'html, ou importés depuis un fichier en .js dans nos pages html.

Les balises html permettant de mettre du JavaScript dans l'html sont:

<script> // ici notre code js </script>

Nous mettrons, du moins au début, ces balises juste avant la fermeture du body en html pour être certain que la page soit chargée avant l'exécution du script.
